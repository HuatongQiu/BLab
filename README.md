## 2. 文件目录结构

每次更新主题后，会将主题相关的资源分享出来，每个主题下将包含多级目录，目录的结构为：

+ 主题
	- docs
	- hardware
	- software
> 解释：1. docs目录下存放此主题所有相关的文档；2. hardware目录下存放的是与主题有关的硬件设计等文档；3. software目录下存放此主题下所有源代码文件
